# 🎨 CSS 动效展示中心 & 交互工坊

一个为个人网站定制的现代、美观且响应式的 CSS 动效分类导航页与实时交互预览平台。目前已集成 42 个前端酷炫动效。

---

## 🌟 核心功能

1. **分类导航 & 检索**
   - **智能分类**：动效自动划分为“导航 & 菜单”、“背景 & 氛围特效”、“卡片 & 图片展示”、“文字 & 排版”、“交互 & 按钮”、“动画 & 视觉效果”等类别。
   - **实时搜索**：支持在顶部检索栏中根据动效名称、序号、分类或代码文件进行模糊搜索。
2. **深浅色主题**
   - 支持深色模式（Dark Mode）与浅色模式（Light Mode）一键切换，并自动将用户偏好保存至 `localStorage`。
3. **多端视口仿真器 (Playground)**
   - 点击动效卡片即可进入独家预览界面。
   - 提供 **Desktop** (桌面端)、**Tablet** (平板端, 768px)、**Mobile** (手机端, 375px) 模拟容器，完美调试响应式布局。
   - 带有精致的虚拟设备物理黑边黑镜与过渡动画。
4. **一键快捷切换**
   - 在预览模式左侧带有“切歌式”目录侧边栏，无需返回大厅即可连续调试不同效果。
5. **本地零跨域支持**
   - 采用 JavaScript 数据集载入方案，直接通过浏览器双击 `index.html`（`file:///` 协议）也能完美运行，不需要启动本地服务器。


## 🚀 运行与使用

### 1. 本地直接预览
直接在任意浏览器中双击 **[index.html](index.html)** 即可开始体验。

### 2. 启动本地静态服务器（可选）
如果您安装了 VS Code，可以使用 `Live Server` 插件运行；或者在项目目录下运行以下命令行快速启动服务：

**使用 Python 快速启动：**
```bash
python -m http.server 8000
```
然后在浏览器中访问 `http://localhost:8000` 即可。

## ➕ 如何新增动效 (自动复用与更新)

当您在未来想要添加新的 CSS 动效时，只需执行以下简单步骤：

### 第一步：将动效拷贝到 effects 目录
将您的动效文件夹拷贝到 `effects/` 目录下。

> ⚠️ **要求**：
> - 文件夹内必须包含一个 `.html` 入口文件（会自动优先匹配 `index.html`，如无则自动选择第一个 `.html` 文件）。
> - 子文件内的 CSS/JS/图片资源请务必使用**相对路径**（例如 `href="style.css"`，不要写成 `/style.css`），以确保其在预览 iframe 中能正确加载。
> - **（可选）** 如果您有该动效的截图，可将其命名为 `[文件夹名].png`（例如 `42-酷炫动效.png`），并存入 `effects/示例效果/` 目录下。

### 第二步：运行更新脚本
在项目根目录下打开命令行终端，运行以下命令：

```bash
python update_effects.py
```

### 第三步：刷新页面
脚本将自动扫描 `effects/` 目录，分类和链接所有文件，并重新生成 `effects_data.js` 数据。此时直接刷新 `index.html` 页面即可看到新添加的动效卡片！
